# @weldr/presets

Project presets and templates package.

## Overview
This package provides project templates and presets for generating new projects with predefined configurations and boilerplate code.

## Key Dependencies
- `handlebars` - Template engine
- `@weldr/shared` - Shared utilities

## Exports
- `.` - Preset generator and utilities

## Commands
- Type check: `pnpm check-types`
- Clean: `pnpm clean`

## Structure
- `src/base/` - Base template configurations
- `src/generator/` - Template generation logic
- `src/generator/templates/` - Handlebars templates

## Features
- Base project templates
- Server-side configurations
- Web frontend templates
- Database setup templates
- Integration templates

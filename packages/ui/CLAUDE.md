# @weldr/ui

UI components library package.

## Overview
This package provides a comprehensive UI component library built with Radix UI primitives and Tailwind CSS, following the shadcn/ui pattern.

## Key Dependencies
- `@radix-ui/*` - Accessible UI primitives
- `tailwindcss` - Utility-first CSS
- `class-variance-authority` - Component variants
- `lucide-react` - Icon library
- `react-hook-form` - Form handling

## Exports
- `./styles/globals.css` - Global styles
- `./lib/*` - Utility libraries
- `./components/*` - UI components
- `./hooks/*` - React hooks
- `./icons` - Icon components

## Commands
- Add component: `pnpm add-component` - Add new shadcn/ui component
- Type check: `pnpm check-types`
- Clean: `pnpm clean`

## Components
- Form components (Input, Button, Select, etc.)
- Layout components (Card, Sheet, Dialog, etc.)
- Data display (Table, Chart, Badge, etc.)
- Navigation (Sidebar, Menu, Breadcrumb, etc.)
- Feedback (Toast, Alert, Progress, etc.)

## Features
- Fully accessible components
- Dark/light theme support
- Customizable with CSS variables
- TypeScript support
- Responsive design

# @weldr/emails

Email templates package using React Email.

## Overview
This package provides email templates built with React Email for transactional emails like password resets and email verification.

## Key Dependencies
- `react-email` - React-based email templates
- `@react-email/components` - Email components
- `tailwindcss` - Styling

## Exports
- `./*` - Email template components

## Commands
- Dev: `pnpm dev` - Start email development server
- Export: `pnpm export` - Export email templates
- Type check: `pnpm check-types`
- Clean: `pnpm clean`

## Templates
- `reset-password.tsx` - Password reset email
- `verification-email.tsx` - Email verification email

## Development
Run `pnpm dev` to start the email preview server at http://localhost:3001
